Cloud Native Platform

Cloud Native PaaS (Platform as a Service)#

In order to keep ATOP blockchain and open finance services stable and resilient on cloud, we developed ATOP Cloud PaaS, named Kupboard. Cloud native is the architecture for assembling all of the cloud-based components in a way that is optimized for the cloud environment. ATOP Cloud Native Platform is the configurable and integrated cloud native platform to develop and operate a variety of services on any cloud infrastructure. ATOP Open Finance Platform, which is resilient and scalable on any cloud, is developed on ATOP Cloud Native Platform to support open finance.

Kupboard is a Cloud Native PaaS that can work on any public cloud or bare-metal servers. The initialization and setup of servers and the build and deployment of applications can be maintained through Kupboard. CLI commands and configurations are easy, but they are very powerful, and customization is also possible depending on the services.

Install

Welcome to the official ATOP repository.

Building from sources#

Supported Systems#

We currently supports the operating systems from the list below:

  • Ubuntu 18.04 or later
  • MacOS 10.14 or later

Prerequisites#

Make sure you have the following utilities installed before building sources.

  • Golang >= 1.15
  • make

Build#

git clone this repository and change directory.

git clone https://github.com/cloudndigital/bc_mainnet.git
cd bc_mainnet

Simply make it!

make install

The binaries assembled (atopnode and atopcli) are located in $GOBIN directory ($HOME/go/bin by default). Make sure you have added $GOBIN to $PATH before you attempt to start ATOP.

That will install the atopcli and atopnode binaries. Verify that everything is working fine.

atopcli version --long

atop-cli should output something similar to:

name: AceTop
server_name: atopnode
client_name: atopcli
version: ""
commit: 3a1f81c0f5290f54a91096a9ebaebba4dcbf519d
build_tags: netgo,ledger
go: go version go1.15.6 darwin/amd64

Build Tags#

Build tags indicate special features that have been enabled in the binary.

  • netgo Name resolution will use pure Go code
  • ledger Ledger devices are supported (hardware wallets)
